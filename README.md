A Hacker News clone where users can post articles.

## Technologies

- **Rails:** 7.1.3.4
- **Ruby:** 3.0.0

### Installation

1. **Clone the repository:**

    ```sh
    git clone git@github.com:vigetlabs/haxxor-news.git
    cd haxxor_news
    ```

2. **Install the required gems:**

    ```sh
    bundle install
    ```

3. **Set up the database:**

    ```sh
    rails db:setup
    ```

4. **Start the Rails server:**

    ```sh
    rails server
    ```
